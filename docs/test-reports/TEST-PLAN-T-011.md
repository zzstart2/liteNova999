## 3. 被测功能模块

### 3.1 主题生命周期管理
- 注册 → 应用 → 切换 → 继承 → 注销
- 多级继承链 (A → B → C)
- 主题切换后渲染器状态同步

### 3.2 渲染器状态一致性
- 配置热更新后变量解析正确
- 多渲染器实例互不影响
- 渲染器与配置对象的引用关系

### 3.3 配置迁移与增量合并
- update_from_dict 增量更新（不丢失未更新字段）
- 多次增量更新的叠加正确性
- 从旧配置迁移到新主题

### 3.4 模板引擎深度测试
- 混合静态文本 + 多变量
- 重复变量出现
- 连续变量无间隔
- 模板中无变量 / 全是变量
- 特殊字符在变量值中

### 3.5 CSS 变量管线
- 5 色 + 2 派生色完整输出
- 主题切换后 CSS 变量同步更新
- 派生色 (hover/disabled) 与主色的对比关系
- 色彩标准化 (HEX/RGB 混用)

### 3.6 多租户场景
- N 个租户并行渲染
- 租户配置修改不互相影响
- 租户共享主题但独立覆盖
