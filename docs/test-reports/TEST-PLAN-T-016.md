## 2. 被测功能模块

### 2.1 DeploymentConfig（部署配置管理）
- 负载均衡配置（策略、权重、健康检查端点）
- 数据库连接配置（主从、连接池、重试）
- 缓存配置（Redis 地址、TTL、前缀、集群模式）
- 监控配置（指标端点、采集间隔、告警阈值）
- 环境标识（production / staging / development）
- 配置校验与合法性验证

### 2.2 LoadBalancer（负载均衡器）
- 轮询 (Round Robin)
- 加权轮询 (Weighted Round Robin)
- 最少连接 (Least Connections)
- 节点健康检查与自动摘除/恢复
- 优雅降级（全部节点不可用时的行为）

### 2.3 DatabasePool（数据库连接池）
- 连接获取与释放
- 连接池大小管理（min/max）
- 连接健康检查与自动淘汰
- 读写分离路由（主节点写、从节点读）
- 重试策略（指数退避）

### 2.4 CacheManager（缓存管理器）
- get / set / delete 基础操作
- TTL 过期机制
- Key 前缀隔离
- 缓存穿透保护（空值缓存）
- 缓存统计（命中率、miss 率）

### 2.5 MonitoringCollector（监控采集器）
- 指标注册与记录（counter / gauge / histogram）
- 健康检查端点（/healthz, /readyz）
- 告警阈值判定
- 指标导出（Prometheus 格式）
