## 1. 测试概述

| 项目 | 内容 |
|------|------|
| **任务编号** | PRJ-LITE999-T-010 |
| **任务名称** | 品牌定制化配置系统 |
| **功能描述** | 可配置品牌系统，Logo/色彩/文案动态替换 |

| **测试日期** | 2025-07-16 |

## 2. 被测功能模块

### 2.1 BrandConfig（品牌配置）
- Logo 配置：URL / 本地路径 / 尺寸约束
- 色彩方案：主色 / 辅色 / 背景 / 文字色（支持 HEX / RGB）
- 文案配置：应用名称 / 标语 / 页脚 / 自定义文案字典
- 支持 JSON/YAML 格式加载与导出

### 2.2 BrandTheme（主题管理）
- 预设主题：default / dark / light
- 主题继承与覆盖
- 色彩自动派生（悬停色、禁用色等）

### 2.3 BrandRenderer（动态渲染）
- 模板文案变量替换 `{{brand.name}}` 等
- Logo URL 解析与 fallback
- 色彩值标准化输出（HEX / RGB / CSS var）

### 2.4 BrandConfigLoader（配置加载器）
- 从 dict / JSON 字符串 / 文件路径加载
- 多层配置合并（默认 → 主题 → 自定义）
- 配置校验（必填项、格式合法性）
